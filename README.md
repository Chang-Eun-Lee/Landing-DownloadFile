# Signal Flow - Download Files

이 저장소는 Signal Flow 자동매매 시스템의 설치 파일을 배포하기 위한 저장소입니다.

## Latest Release

최신 버전의 설치 파일은 [Releases](https://github.com/Chang-Eun-Lee/Landing-DownloadFile/releases) 페이지에서 다운로드하실 수 있습니다.

## 설치 방법

1. [Releases](https://github.com/Chang-Eun-Lee/Landing-DownloadFile/releases) 페이지에서 최신 버전의 `.exe` 파일 다운로드
2. 다운로드한 설치 파일 실행
3. 설치 마법사의 지시에 따라 설치 진행
